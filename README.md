# MadinMD

Aplikasi manajemen data murid TPQ/Madrasah Diniyah berbasis **Flask** dan **PostgreSQL**.

## 🚀 Fitur
- Login Admin
- CRUD Data Murid (Tambah, Edit, Hapus)
- Biodata Murid
- Input Nilai & Deskripsi Otomatis
- Export CSV & PDF

## 🛠️ Teknologi
- Python (Flask)
- PostgreSQL (psycopg2)
- Bootstrap (Frontend)
- FPDF (Export PDF)
